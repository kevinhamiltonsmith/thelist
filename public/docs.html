About TheList

Steve Koepke
Steve compiles a list of Northern California concerts of all sizes and genres. Originally focusing on funk, punk, ska, and thrash shows, the list now represents a plethora of genres and shows, including some not available on any other website. For more information or to submit events, please contact Steve via his personal site: Steve's Site.

Format of Events
The events are returned with a standard format as follows:
eventID : ObjectId. A unique identifier for each event.
date : [Date]. A list of dates the event is occurring on. Dates are stored in ISO format.
artists : [String]. A list of all artists by name.
venue : String. Contains the name of the venue.
address : String. The address field contains a city and, when provided, a street address.
ages : String. Indicates ages that are allowed to attend.
specialInfo : [String] Special information about the event.
pitWarning : Boolean. True if there is a chance of a mosh pit.
willSellout : Boolean. True if event has a high risk to sell out.
noInsOuts : Boolean. True if no ins and outs allowed at event.
underagePayMore : Boolean. True if underage attendees must pay more for the event.
priceAndTime : String. If price and time are given, this field will contain that information.
recommended : Number from 0-3 inclusive. Indicating Steve's reccomendation of event
